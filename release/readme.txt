ACNL_PIC_Changer

使い方
1.ACNL_PIC_Changer.exeを実行
2.「select save data」を押してセーブデータ(garden_plus.dat)を選択
3.「select pic data」を押して差し替えたい画像データ(64×104のJPG)を選択
4.「start」を押して差し替えスタート
5.保存ダイアログがでるので任意の場所に保存
6.保存したデータをACNL Save Editorにアップロードして何もしないで保存(セーブデータの修復)
7.セーブデータをとびだせ どうぶつの森に書き戻す
以上
